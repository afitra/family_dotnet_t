## Installation

#### Manual instalation project dotnet:

    1. clone repository   
    3. setup appsettings.json file
    4. run 'dotnet restore'
    5. eksekusi file query di folder "query" berurutan sesuai nomor
    6. run 'dotnet watch'
    7. enjoy with dotnet API  /swagger/index.html
